# TrainSmart--AI-Based-Train-Selection-and-Route-Analytics-System
TrainSmart AI is an ML-based train recommendation system that suggests optimal trains using fare, duration, stops, and distance. Built with Streamlit and Scikit-learn, it includes analytics dashboards, fare prediction, user authentication, and personalized features using SQLite.

How to Run the Project
1. Create a virtual environment|
eg python -m venv venv
2. Activate the virtual environment |
Windows:
venv\Scripts\activate |
Mac/Linux:
source venv/bin/activate |
3. Install required libraries
|pip install -r requirements.txt
4. Run the application
|streamlit run App.py
